🚀 TaskFlow: Professional Productivity Suite
TaskFlow is a modern, high-performance To-Do application designed for users who value speed, aesthetics, and organization. Built with a "Carbon-Dark" design language, it provides a distraction-free environment to manage daily goals effectively.

🌟 Created and Designed by Madhav Soni (BITS Goa)
✨ Key Features
Priority-Based Management: Organize tasks using High 🔴, Medium 🟡, and Low 🟢 priority levels.

Dynamic Progress Tracking: A real-time productivity bar that calculates your completion percentage.

Smart Filtering: Instantly toggle between All, Pending, and Completed views.

Persistent Storage: Automatically saves your tasks to localStorage so you never lose your data.

Responsive Design: Fully optimized for Desktop, Tablet, and Mobile devices.

Intelligent UI: Features subtle animations, glassmorphism effects, and intuitive icons powered by Lucide.

🛠️ Tech Stack
HTML5: Semantic structure for accessibility.

CSS3: Custom variables, Flexbox/Grid layout, and smooth transitions.

JavaScript (Vanilla): High-efficiency DOM manipulation and logic.

Lucide Icons: Professional-grade iconography.

🚀 Getting Started
TaskFlow is a zero-dependency project. You don't need to install anything.

Download the index.html file.

Open it in any modern web browser (Chrome, Safari, Firefox, or Edge).

Start crushing your goals!

⌨️ Keyboard Shortcuts
Enter: Quickly add a task while the title input is focused.

Tab: Seamlessly navigate through inputs and filters.

📸 Design Philosophy
The application follows a Minimalist Dark aesthetic. By using a deep palette (#09090b), it reduces eye strain during long working hours. The blue accent (#3b82f6) provides a professional "Dev-Tool" feel, while the priority indicators ensure that your most important work never gets lost in the noise.
